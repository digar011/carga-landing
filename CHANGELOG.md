# Changelog — CarGA Landing Page

## [1.0.0] — 2025-03-24

### Added
- Production landing page (`index.html`) with all 6 sections: hero, problem, how it works, stats, signup, footer
- Waitlist signup form with client-side validation and localStorage persistence
- Signup counter (base 47 + actual signups)
- Responsive mobile-first design (navy/gold/white palette)
- Sticky navbar with backdrop blur on scroll
- Smooth scroll CTA from hero to signup section
- SEO meta tags, Open Graph, Twitter Cards, JSON-LD structured data
- Interactive 8-screen mobile app prototype (`prototype.html`) for investor demos
- Playwright E2E tests for both landing page and prototype
- Full project documentation: CLAUDE.md, README.md, ONBOARDING.md, PRODUCT.md, TODO.md, .env.example
